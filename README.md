Here's the problem: Given a list of numbers, return a list of all the numbers that are unique within that list (i.e. they aren't duplicated).

Let's clarify this with an example:

Input: [1, 10, -4, 2, 7, 8, -2, 98, 10, 8, 2]

Output: [1, -4, 7, -2, 98]
